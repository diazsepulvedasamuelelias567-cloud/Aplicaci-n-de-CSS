Tipos de redes: LAN, MAN y WAN

Las redes informaticas permiten compartir informacion, recursos y servicios. Segun su alcance geografico, pueden clasificarse en LAN, MAN y WAN.

Concepto general

Una red es un conjunto de dispositivos conectados entre si para intercambiar datos. El tipo de red depende de la distancia que cubre y del numero de equipos que puede interconectar.

L

LAN

Local Area Network o red de area local.

Conecta equipos dentro de un espacio pequeno, como una casa, aula, oficina o laboratorio.

  • Alcance corto
  • Alta velocidad
  • Uso domestico o escolar
M

MAN

Metropolitan Area Network o red de area metropolitana.

Une varias redes locales dentro de una ciudad o una zona urbana amplia.

  • Cobertura intermedia
  • Conecta sedes cercanas
  • Uso institucional o empresarial
W

WAN

Wide Area Network o red de area amplia.

Interconecta redes a grandes distancias, incluso entre paises o continentes.

  • Alcance muy grande
  • Puede usar internet
  • Ejemplo principal: la red mundial

Comparacion rapida

Tipo Alcance Ejemplo

Administracion de una LAN

En una red local es comun conectar computadores, routers y switches dentro de un mismo espacio.

Conectividad para MAN y WAN

La velocidad de transmision y la estabilidad son claves cuando una red cubre mayores distancias.

Ejemplo de uso

En una escuela, la sala de informatica puede funcionar como una LAN. Si varias sedes del colegio se conectan dentro de la misma ciudad, forman una MAN. Y si esa institucion comparte informacion con sedes en otros paises, ya participa en una WAN.
